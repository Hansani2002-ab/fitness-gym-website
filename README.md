# 🏋️ Fitness Pro - Gym Promotional Website

## 📌 Project Overview

Fitness Pro is a modern and responsive gym promotional website built to showcase services, trainers, and membership plans while allowing users to contact the gym easily. The design focuses on clean UI/UX and smooth user interactions.

---

## 🚀 Tech Stack

* ⚡ Next.js
* 🎨 Tailwind CSS
* 📧 EmailJS (Contact form integration)
* 🔐 Clerk (Authentication)
* 🎬 Framer Motion (Animations)

---

## ✨ Features

* Fully responsive design (Mobile, Tablet, Desktop)
* Modern UI with dark & light theme
* Animated sections using Framer Motion
* Contact form with EmailJS integration
* Clean and reusable component structure
* Optimized performance with Next.js

---

## 🛠️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/Hansani2002-ab/fitness-gym-website

```

2. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```

4. Open in browser

```bash
http://localhost:3000
```
